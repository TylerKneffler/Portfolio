# Tyler Kneffler - Portfolio

A modern React-based portfolio showcasing full-stack development projects and interactive experiences.

## 🌟 Features

- **Interactive 2D Galaxy Engine**: Custom-built Canvas-based galaxy visualization system
- **Responsive Design**: Optimized for all device sizes
- **Modern Tech Stack**: React, Three.js, and modern web technologies
- **Performance Optimized**: Fast loading and smooth animations

## 🚀 Live Demo

Visit the portfolio at: [https://tylerkneffler.github.io/Portfolio](https://tylerkneffler.github.io/Portfolio)

## 🛠️ Tech Stack

- **Frontend**: React 19+, React Router
- **Graphics**: Three.js, HTML5 Canvas
- **Styling**: CSS3, Responsive Design
- **Deployment**: GitHub Pages with GitHub Actions
- **Build Tools**: Create React App, npm

## 📦 Installation & Development

```bash
# Clone the repository
git clone https://github.com/TylerKneffler/Portfolio.git

# Navigate to project directory
cd Portfolio

# Install dependencies
npm install

# Start development server
npm start
```

## 🚀 Deployment

This portfolio is automatically deployed to GitHub Pages using GitHub Actions. Any push to the `main` branch will trigger a new deployment.

# Build
npm run build

# Deploy
npm run deploy
